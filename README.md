## How it works

Inside `const.py` you can change:

```
LIMIT=? # articles limit
TOPIC=? # python | nextjs | ect...
```

## Installation

1. Install the `bs4` (Beatiful soup v4) library

```
pip install bs4
```

2. Run the app with:

```
python src/main.py
```
